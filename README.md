# ChatFix

Μικρή σελίδα (static) για να κάνεις paste κείμενο από ChatGPT και να το “καθαρίζεις” (bullets, markdown, intro/disclaimer, long dashes, quotes) ώστε να φαίνεται πιο φυσικό, με ένα click για copy.

## Local preview

Άνοιξε το `index.html` σε browser (κάποιες λειτουργίες clipboard μπορεί να είναι περιορισμένες σε `file://`).

## GitHub Pages (δωρεάν)

1. Κάνε push τα αρχεία στο GitHub.
2. Repo → **Settings** → **Pages**
3. **Build and deployment** → **Deploy from a branch**
4. Διάλεξε branch `master` και folder `/ (root)` → **Save**

Μετά από λίγο θα πάρεις URL τύπου `https://<user>.github.io/<repo>/`.
